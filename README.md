# Proposal
## Overview
Analyze expected floor depth reached and ROI per run given user stats and strategy
## Assumptions
- Gain bomb and food by buying
- No combat except monsters attack player, when combating, the player and the monster take turns to attack each other. (There is certain monster types that actively attack player.)
## Hypothesis
1. Higher luck increases expected floor depth more than bomb usage
2. No-food and bomb strategy produces higher average net profit in low-risk runs but lower net profit in high-risk runs.
3. Deeper floor reached, higher net profit gained. (cuz deeper, more expensive the ore will gain)

## Phases
### Phase 1 - Random variables
- Ladder probability
- Shaft depth distribution
- Resource drop rates
- Monster spawn probability
- Player damage / survival

### Phase 2 - Controls & experiments
Player stats including:
- Luck level
- Mining skill level
- Bombs vs Pickaxe strategy
- Food vs No food
### Phase 3 - Analysis
- Expected floor reached
- Expected ores per run
- Survival probability
- Net profit for different strategy
